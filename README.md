# SteamHunter成就查询

快速查询Steam成就信息并通过机器人播（zhuang）报（bi）的简易插件。

使用方法：
1、查steam成就 SteamID/Steam64ID/Steam主页URL

ps：需要保持Steam个人资料公开并在SteamHunter（steamhunters.com）网站更新账号信息（点击账户头像旁的update）。
